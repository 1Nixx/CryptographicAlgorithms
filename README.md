# CryptographicAlgorithms
WPF desktop application that allows you to test various encryption and digital
signature algorithms

# Features
- Saving and loading from files
- Elementary ciphers:
  - Column Method
  - Decimation Method
  - Vigenere Method
- LFSR Method
- Elgamal Method
- Digital signature based on RSA
